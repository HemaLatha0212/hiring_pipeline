
# Hiring Pipeline Challenge

## Problem Statement
Small engineering teams can't afford ATS platforms like Greenhouse or Lever. They manage hiring pipelines on spreadsheets, losing track of applicants and manually chasing people who stopped responding.

### The Challenge
Build a lightweight internal tool that manages a job application pipeline without spreadsheets.  
- Companies define active capacity.  
- Applications beyond capacity enter a waitlist.  
- Automatic promotion when slots free.  
- Applicants can check status and queue position.  
- Handle concurrency and inactivity decay.
